FRAMEWORK

what is a Framework...?

A framework is a platform for developing software applications. It provides a foundation on which software developers can build programs for a specific platform.

A framework is important because

  • It makes programming much easier.
  • With the help of a framework, a coder gets basic functionalities and with the help of it one can focus on high-level functionalities.
  • It also reduces time to develop applications.
  • It makes easier to work on sophosticated technologies.
  • It is reliable and secure.
